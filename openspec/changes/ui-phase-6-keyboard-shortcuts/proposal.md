# Phase 6: Keyboard Shortcuts

## Intent

Add comprehensive keyboard shortcuts for power users to work 3x faster without mouse.

## Scope

- `Ctrl+F` — Focus search input
- `Ctrl+Shift+S` — Start scan
- `Ctrl+R` — Recommend playlist
- `Ctrl+E` — Export to Serato
- `Ctrl+M` — Toggle Missing column
- `Space` — Preview selected track (already exists)
- `Enter` — Open selected track in default player
- `Delete` — Remove selected track from playlist

## Success criteria

1. All shortcuts work in their respective contexts
2. Shortcuts don't conflict with existing shortcuts
3. All tests pass
4. All verification commands pass
