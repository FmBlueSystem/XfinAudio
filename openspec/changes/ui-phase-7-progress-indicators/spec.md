# Specification: Progress Indicators

## Requirements

### R1. Scan progress

**GIVEN** a scan operation is in progress
**THEN** a progress bar is visible showing percentage and estimated time

### R2. Recommend progress

**GIVEN** a recommendation operation is in progress
**THEN** a progress bar is visible showing percentage and estimated time

### R3. Export progress

**GIVEN** an export operation is in progress
**THEN** a progress bar is visible showing percentage and estimated time
