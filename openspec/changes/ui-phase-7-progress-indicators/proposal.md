# Phase 7: Progress Indicators

## Intent

Add visual feedback for long operations (scan, recommend, export) so users know the app is working.

## Scope

- Add a progress bar to the scan button during scanning
- Add a progress bar to the recommend button during recommendation
- Add a progress bar to the export button during export
- Show estimated time remaining (e.g., "2:30 remaining")

## Success criteria

1. Progress bars are visible during operations
2. Progress bars show percentage and estimated time
3. All tests pass
4. All verification commands pass
