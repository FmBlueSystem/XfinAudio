# Specification: Space Utilization

## Requirements

### R1. Table expansion

**GIVEN** the Library screen is shown
**WHEN** the window is resized vertically
**THEN** the track table expands to fill available space

### R2. Minimum height

**GIVEN** the Library screen is shown
**THEN** the track table has a minimum height of 400px (not 132px)

### R3. No maximum height

**GIVEN** the Library screen is shown
**THEN** the track table has no maximum height constraint (not 190px)
