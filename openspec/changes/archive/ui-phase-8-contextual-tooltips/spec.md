# Specification: Contextual Tooltips

## Requirements

### R1. Button tooltips

**GIVEN** any button in the UI
**THEN** it has a helpful tooltip explaining its function

### R2. Table column tooltips

**GIVEN** any table column header
**THEN** it has a tooltip explaining the column

### R3. Help button

**GIVEN** a complex feature
**THEN** a "What's this?" help button opens a help dialog

### R4. Tour button

**GIVEN** the main window
**THEN** a "Tour" button shows a step-by-step walkthrough
