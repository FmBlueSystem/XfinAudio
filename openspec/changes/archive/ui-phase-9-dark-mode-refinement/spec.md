# Specification: Dark Mode Refinement

## Requirements

### R1. Text contrast

**GIVEN** the dark theme is applied
**THEN** all text has at least 4.5:1 contrast ratio against background

### R2. Button gradients

**GIVEN** the dark theme is applied
**THEN** buttons have subtle gradients for depth

### R3. Hover states

**GIVEN** the dark theme is applied
**THEN** all interactive elements have hover states

### R4. Focus outlines

**GIVEN** the dark theme is applied
**THEN** focusable elements have focus outlines for accessibility
