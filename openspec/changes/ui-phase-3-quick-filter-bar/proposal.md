# Phase 3: Quick Filter Bar

## Intent

Add persistent quick filters above the track table so users can filter without switching to the Metadata Worklist tab.

## Scope

- Add a `QLineEdit` for search (make it more prominent)
- Add filter chips/buttons for: Complete/Incomplete, Missing BPM, Missing Key, Missing Energy
- Add a "Clear Filters" button
- Show active filter count in a badge

## Success criteria

1. Filter bar is visible above the track table
2. Filter buttons toggle and highlight active filters
3. Clear button resets all filters
4. All tests pass
5. All verification commands pass
